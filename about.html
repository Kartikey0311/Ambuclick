Details about the HTML and CSS Code

The HTML and CSS code for AmbuClick is designed to provide a seamless and visually appealing experience for users. HTML (Hypertext Markup Language) structures the content of the web pages, while CSS (Cascading Style Sheets) adds styling to enhance the overall look and feel of the website. The code follows best practices for web development to ensure compatibility and responsiveness across various devices.Slight javascript was used but the major part is covered by html and css

Meet the Creators

AmbuClick is the result of the collaborative efforts of Kartikey Aggarwal and Tanish Kini. Both creators are passionate engineers in the Electronics and Telecommunication (EXTC) department. Their dedication to leveraging technology for emergency healthcare accessibility is reflected in the design and functionality of AmbuClick.

Challenges Faced

During the development of AmbuClick, the creators faced various challenges, including ensuring a user-friendly interface, addressing cross-browser compatibility issues, and optimizing the website for performance. Through perseverance and innovation, we overcame these challenges to deliver a reliable and efficient platform for emergency healthcare.The major challenge faced by us was when we tried to complete the code together , sharing of files became an issue for us as we had no prior knowledge of github . We wanted to have a experience similar to canva where one can design together with his/her team members.
